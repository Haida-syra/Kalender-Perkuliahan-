# Kalender-Perkuliahan-
Produk berbasis Microsoft Excel berupa kalender perkuliahan untuk mahasiswa, dirancang agar jadwal kuliah, tugas, dan ujian tercatat rapi serta mudah dipantau. Fitur input jadwal, pengingat otomatis, dan visualisasi kalender interaktif membantu mahasiswa mengatur waktu, mengurangi risiko lupa, dan meningkatkan efektivitas belajar.
